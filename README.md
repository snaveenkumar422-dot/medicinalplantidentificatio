Medicinal Plant Identification Using Image Classification
Deep Learning | CNN | Python | TensorFlow/Keras

This project aims to identify medicinal plant species from images using deep learning-based image classification.
A Convolutional Neural Network (CNN) model is trained on a dataset of medicinal plant images to accurately predict the plant species and provide potential therapeutic benefits.
